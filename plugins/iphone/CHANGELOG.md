# Changelog

## 0.1.0 - 2026-07-18

- Initial release, replacing the earlier `imessages` and `iphone-history`
  placeholder plugins with one full-featured plugin.
- Read iMessage history: browse conversations, search across all of them,
  and pull up a specific chat's messages, including photos/files attached to
  a message.
- Send new iMessages, with a preview step that always shows the resolved
  recipient and exact text before anything is actually delivered.
- Maintain an optional trusted-contacts list so message reads can flag
  senders you already trust.
- Read Mac/iPhone call history: cellular calls (via Continuity) and FaceTime
  Audio/Video, filterable by date, contact, direction, or missed-only.
- Look up Contacts by name or number, shared across the messages and calls
  tools.
